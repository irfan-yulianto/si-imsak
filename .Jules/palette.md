## 2024-03-24 - Search Input UX Improvement
**Learning:** Adding a clear button (XIcon) inside search inputs that conditionally renders when the query length is > 0 is a standard, preferred micro-UX pattern for this application's interface. Ensure to add adequate right padding (e.g., `pr-8`) to prevent text overlap with the icon.
**Action:** Always implement a clear button functionality for search inputs to improve UX.
