## 2024-05-18 - Input Clear Button Pattern
**Learning:** For small/narrow inputs commonly used for searching or filtering on mobile, users can find it difficult to manually backspace the entire string if they change their mind. Providing a small, absolutely positioned clear button ("X" icon) inside the right side of the input that dynamically appears when input text exists is a well-received micro-UX enhancement that prevents frustration.
**Action:** Always consider adding a conditionally rendered "X" clear button inside custom search inputs, particularly ensuring it has sufficient touch target size, hover states, and an `aria-label` for screen reader accessibility. Ensure to add padding on the right side of the input (e.g., `pr-8`) so the text does not overlap with the absolutely positioned icon.
